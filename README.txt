# Análisis de Datos de la Inflación en Panamá (1960-2023)

- Descripción

Este proyecto realiza un análisis de datos sobre la inflación en Panamá desde 1960 hasta 2023. Utiliza un archivo CSV que contiene su respectiva fecha y porcentaje de inflación anual. Se han implementado varias visualizaciones y análisis estadísticos para explorar y comprender los datos.

- Requisitos (Incluido en requirements.txt)

- Python 3.8 o superior
- Paquetes de Python:
  - pandas
  - matplotlib
  - scipy
  - Pillow
  - tkinter

- Instalación

1. Clona el repositorio:
   ```bash
   git clone https://github.com/camiloohr/datasetinflacion.git

2. Navega al directorio de tu repositorio:
cd datasetinflacion

3. Instala los paquetes necesarios:
pip install pandas matplotlib scipy Pillow

4. Ejecuta el script principal:
Puedes utilizar la consola local para ejecutar con:
python proyecto_final.py
O en su defecto, puedes utilizar sencillamente el entorno de tu preferencia para ejecutar codigo de Python. (Se recomienda Visual Studio Code)

IMPORTANTE: Ejecutar el script desde la carpeta del repositorio descargado o clonado, ya que es necesario que los archivos se encuentren juntos para que el programa funcione correctamente.

- Funcionalidades

Visualizaciones:
Histograma: Muestra la distribución de los porcentajes de inflación.
Gráfico de Dispersión: Visualiza la inflación a lo largo del tiempo.
Diagrama de Caja: Muestra la dispersión y los valores atípicos de los datos de inflación.
Línea de Tiempo: Presenta la evolución de la inflación a lo largo del tiempo.

Controladores:
Mostrar Inflación por Década: Permite seleccionar una década específica (1960s, 1970s, ..., 2020s) y muestra una lista con los porcentajes de inflación de cada año de esa década.
Mostrar Gráfica por Década: Permite seleccionar una década específica y muestra un gráfico de la inflación durante esa década.

Menú Principal:
Presentación: Muestra una ventana con la información de la presentación del proyecto.
Ver Histograma: Muestra el histograma de la inflación.
Ver Gráfico de Dispersión: Muestra el gráfico de dispersión de la inflación.
Ver Diagrama de Caja: Muestra el diagrama de caja de la inflación.
Ver Línea de Tiempo: Muestra la línea de tiempo de la inflación.
Ver Histograma Filtrado por Años: Muestra un histograma de la inflación filtrado por un rango de años.
Ver Histograma Filtrado por Inflación: Muestra un histograma de la inflación filtrado por un rango de valores de inflación.
Mostrar Inflación por Década: Permite seleccionar una década y muestra los porcentajes de inflación de cada año de esa década.
Mostrar Gráfica por Década: Permite seleccionar una década y muestra un gráfico de la inflación de esa década.
Salir: Cierra la aplicación.
